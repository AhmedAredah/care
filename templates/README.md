# Templates

Each `*.yaml` file in this directory is a single crash-report template
recognised by the template detector (`care/templates/`).

The schema is defined in `care/templates/schemas.py`. See
`docs/template-authoring.md` (Phase 7) for a walk-through.

`templates/example_state/example_template_v1.yaml` is the reference
example — synthetic, not based on any real DOT form.
